BayernAtlas Web Component Examples

Quick Start

1. Import the Web Component via <script> Tag

HTML:

<script src="https://atlas.bayern.de/wc.js" type="module" crossorigin="anonymous"></script>

2. Add the <bayern-atlas> Tag

HTML:

<bayern-atlas></bayern-atlas>

3a. Setup Map Via Attributes

HTML:

<bayern-atlas l="GEORESOURCE_AERIAL" z="10" c="11,48" ec_map_activation="false">

3b. Setup Map programmatically

HTML:

<bayern-atlas l=""></bayern-atlas>

JavaScript:

document.querySelector('bayern-atlas') // register a load-event listener on the map
	.addEventListener('baLoad', (event) => { // save to call the map now
		const baMap = event.target;
		baMap.addLayer(baMap.GEORESOURCE_AERIAL); // add a layer to the map
		baMap.modifyView({ zoom: 10, center: [11, 48] }); // position the map
	});
